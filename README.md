# STM32H745 Dual-core M4-M7

Проект представляет собой тестовую прошивку для микроконтроллера STM32H745 на языке C (bare-metal). В рамках задания реализуются три задачи: управление светодиодами, взаимодействие двух ядер через разделяемую память и настройка таймера.

## Статус выполнения

| Задача | Описание | Статус |
|--------|----------|--------|
| 1 | Мигание LED и управление вторым LED через кнопку User Button | ✅ Выполнено |
| 2 | Реализация разделяемой памяти и взаимодействие двух ядер | ⏳ В процессе |
| 3 | Реализация таймера с частотой 40-80 кГц | ⏳ Запланировано |

## Задача 1: Управление светодиодами

### Описание
- Реализовано мигание одного светодиода (LED1) с фиксированной частотой
- Второй светодиод (LED2) зажигается по нажатию на кнопку User Button
- Проект написан на bare-metal (без использования HAL или CMSIS)

### Используемые ресурсы
| Периферия | Назначение |
|-----------|------------|
| GPIOA | Управление светодиодами (LED1, LED2) |
| GPIOC | Чтение состояния кнопки User Button |
| RCC | Тактирование портов ввода-вывода |
| SysTick | Таймер для задержек (опционально) |

### Схема подключения (STM32H745 Discovery Kit)
| Компонент | Порт | Пин |
|-----------|------|-----|
| LED2 (User LED2) | PB | 1 |
| LED3 (User LED3) | PA | 14 |
| User Button | PC | 13 |

*Примечание: точное назначение пинов может отличаться в зависимости от используемой отладочной платы.*

### Принцип работы
1. При включении микроконтроллера LED1 начинает мигать с частотой 1 Гц
2. При нажатии кнопки User Button происходит смена состояния LED2 (вкл/выкл)
3. Для обработки нажатия кнопки используется программный антидребезг
